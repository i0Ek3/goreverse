> Reverse multi values in Go.

## Getting Started

### Installation

`go get https://github.com/i0Ek3/goreverse`

### Import

```Go
import (
    gr "github.com/i0Ek3/goreverse"
)

func main() {
    b := []byte{}
    gr.Reverse(b, 0, 2)
}

```
